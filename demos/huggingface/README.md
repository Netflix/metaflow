# Hugging Face decorator demo

Runnable flows and **`run_huggingface_demo.py`** (argparse CLI). **[docs/huggingface.md](../../docs/huggingface.md)** § Demo documents **defaults**, a **step-by-step “How to test each use case”** table (what is being tested and the exact command), and **using your own models**.

Quick start from repo root (defaults: public Hub metadata, no token):

```bash
./demos/huggingface/run_huggingface_demo.sh run --help
./demos/huggingface/run_huggingface_demo.sh run
```

Pass flags after `run` (see `--help` epilog for defaults and built-in repos).
